## Install Packages

```
sudo apt install apache2
```

## Adjust Firewall

```
sudo ufw allow 80
sudo ufw allow 443
```

## Install PHP Packages

```
sudo apt install php7.2 libapache2-mod-php7.2 php7.2-mysql php7.2-gd php7.2-opcache
```
